 Two-Factor Authentication System (Python)

This project implements a 2FA login system using:
- Username & Password
- OTP (One-Time Password)
- Email delivery

Technologies:
- Python
- pyotp
- SMTP Email

Purpose:
Enhances login security by adding an extra authentication layer.

⚠️ Security Notice

This project uses placeholder email credentials.

Before running:
1. Enable 2-Step Verification on your Gmail.
2. Generate a Gmail App Password.
3. Replace:
   - sender_email
   - sender_password
   - receiver_email

Never upload real passwords to GitHub.
